Tags

You include HaXtatic tags in your content sources with a simple notation: often {Prefix{Name}}, sometimes {Prefix{Name: Args}}. (Why curly braces and not <angle brackets>? Easier authoring in all tools and naturally prompter processing.)

Example: {P{Date}} includes the current 'page date': 2016/12/18. Supported 'prefixes' or tag types are T, P, B, X:

  • T-tags: named plain text/markup snippets

    {T{..}} tags directly include named text values that you defined once in your haxtatic.config and then reuse wherever needed.

  • P-tags: per-page values (date, title, custom..)

    {P{..}} tags include (built-in and custom) values specific to the current page.

  • B-tags: blok info (name, title, desc)

    {B{..}} tags include (named or current) blok descriptors.

  • X-tags: custom renderers

    {X{..}} tags include a variety of configurable programmatic renderers.
