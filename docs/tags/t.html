T-tags: named plain text/markup snippets

{T| .. |} tags directly output named textual contents defined in a *.haxproj file.

In a nutshell

  • Definitions in *.haxproj:
    |T|AuthBlurb: The author is a serial enthusiast
    role-playing <a>on XingedIn</a>.
  • Usage anywhere: {T|AuthBlurb|}

See also

  • To define page-specific / unique-per-page values (still fully referable from outside the page ie. in templates, snippets and *.haxproj files), use custom P-tags.
  • For more versatile parameterized snippets with named variable value substitutions, declare an X-tag definition (or multiple) of type hax.snippet instead.
