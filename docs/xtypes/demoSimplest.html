demoSimplest

Outputs <h1>Hello World!</h1>.

In a nutshell

  1. Declaration of an |X|demoSimplest:myTag directive in a *.haxproj file,
    • further configuring: (no settings)
  2. Usage anywhere: {X|myTag |}

Stage:

Early — this X-renderer does not require a page context for rendering: therefore any demoSimplest-invoking X-tags encountered during pre-templating (at start-up time) will be processed eagerly (for overall-reduced per-page processing load).

Purpose

This "verging on useless" X-renderer exists solely to serve as a code sample to start from when wanting to develop custom X-renderers within the HaXtatic code-base in Haskell.
