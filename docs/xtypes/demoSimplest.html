hax.demoSimplest

Outputs <h1>Hello World!</h1>.

In a nutshell

  1. Declaration of an |X|hax.demoSimplest:myXtag directive in a *.haxproj file,
    • further configuring: (no settings)
  2. Usage anywhere: {X|myXtag |}

This "slightly pointless" X-renderer exists solely to serve as a code sample to start from for Haskell programmers wanting to develop custom X-renderers within the HaXtatic code-base.
