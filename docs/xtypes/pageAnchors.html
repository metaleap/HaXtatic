hax.pageAnchors

Outputs a sequence of <outerhtmltag ..><a ..> .. </a></outerhtmltag>, one per <gatherhtmltag> element found in the page's content source.

In a nutshell

  1. Declaration of an |X|hax.pageAnchors:myTag:gatherhtmltag: .. directive in a *.haxproj file,
    • further configuring: considerEmpty, htmlIfEmpty
  2. Usage anywhere: {X|myTag:outerhtmltag: htmlAtts = [..] |}

Stage:

Page — this X-renderer requires a page context for rendering: therefore any hax.pageAnchors-invoking X-tags encountered during pre-templating (at start-up time) will be skipped over and thus delayed until page time.
