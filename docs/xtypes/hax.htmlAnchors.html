hax.htmlAnchors

Outputs a sequence of  <outerhtmltag ..><a href="#.." ..> .. </a></outerhtmltag>, one per  <gatherhtmltag> element found in the page's content source.

In a nutshell

  1. Declaration of an |X|hax.htmlAnchors:myTag:gatherhtmltag: .. directive in a *.haxproj file,
    • further configuring: considerEmpty, outputIfEmpty, xmlEscapeHref
  2. Usage anywhere: {X|myTag:outerhtmltag: attr = [..] |}

Stage:

Page — this X-renderer requires a page context for rendering: therefore any hax.htmlAnchors-invoking X-tags encountered during pre-templating (at start-up time) will be skipped over and thus delayed until page time.
