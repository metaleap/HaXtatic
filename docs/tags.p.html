P-tags

{P{...}} tags include values specific to the current page, both custom and predefined.

{P{Title}} — page title

Includes in its place the current page's <h1> HTML tag's entire inner markup (the first one found within a single line, that is), if any (otherwise empty). Useful for use in your template's <title> HTML tag.

Example: here's how the <head> of this site's theme.tmpl.html (and thus that of every single output page) begins:

<title>{P{Title}} | {T{SiteTitle}}</title>
