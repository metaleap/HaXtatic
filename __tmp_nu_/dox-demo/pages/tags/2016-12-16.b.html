B-tags: blok info (name, title, desc)

{B|..|} tags include (named or current) {T| HaxBlokLink |} properties.

If a name is not supplied, all these refer to the current (if any) blok:

  • {B|name:name|} → name of the named or current {T| HaxBlok |} page's parent {T| HaxBlok |}
    — eg. for this page: {B|name|}
  • {B|title:name|} → title of the named or current {T|HaxBlok|}
    — eg. in this site for name basics: {B|title:basics|}
  • {B|desc:name|} → description of the named or current {T|HaxBlok|} — eg. in this site for this page:
  • {B|atomFile:name|} → relative path to the autogenerated .atom file (if any) for the named or current {T|HaxBlok|}
  • {B|blokIndexPageFile:name|} → relative path to the autogenerated index page for the named or current {T|HaxBlok|}
    — eg. in this site for name basics: {B|blokIndexPageFile:basics|}
  • {B|dtFormat:name|} → name of the preferred date-time format (if any) for the named or current {T|HaxBlok|}
