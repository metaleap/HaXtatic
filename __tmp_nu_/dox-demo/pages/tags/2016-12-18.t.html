T-tags: named plain text/markup snippets

{T| .. |} tags directly output named textual contents defined in a {T|HaxConfLink|} file.

{X|h: {T|Briefly|} |}
  • Definitions in {T|HaxConfLink|}:
    |T|AuthBlurb: The author is a serial enthusiast
    role-playing <a>on XingedIn</a>.
  • Usage anywhere: {T|AuthBlurb|}
  • Leading/trailing white-space is stripped of both the name and the value.
{X|h: See also |}
  • To define page-specific / unique-per-page values (still just as referable from outside the page ie. in templates, snippets and other {T|HaxConfLink|} definitions), use custom {T|HaxPTag|}s.
  • For more versatile parameterized snippets with custom variable substitutions, use X-tags of type hax/snippet instead.
