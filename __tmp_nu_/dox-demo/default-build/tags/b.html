B-tags: blok info (name, title, desc)

{B|..|} tags include (named or current) blok properties.

If a name is not supplied, all these refer to the current (if any) blok:

  • {B|name:name|} → name of the named or current blok page's parent blok
    — eg. for this page: tags
  • {B|title:name|} → title of the named or current blok
    — eg. in this site for name basics: Basics
  • {B|desc:name|} → description of the named or current blok — eg. in this site for this page:
  • {B|atomFile:name|} → relative path to the autogenerated .atom file (if any) for the named or current blok
  • {B|blokIndexPageFile:name|} → relative path to the autogenerated index page for the named or current blok
    — eg. in this site for name basics: basics/index.html
  • {B|dtFormat:name|} → name of the preferred date-time format (if any) for the named or current blok
