B-tags: blok info (name, title, desc)

{B|..|} tags include (named or current) blok descriptors.

  • {B|name|} → name of the current blok page's parent blok
    — eg. for this page: tags
  • {B|title|} → title of the current blok page's parent blok
    — eg. for this page: Tags
  • {B|desc|} → description of the current blok page's parent blok — eg. for this page:
  • {B|title:name|} → title of the named blok
    — eg. in this site for name basics: Basics
  • {B|desc:name|} → description of the named blok — eg. in this site for name basics:
