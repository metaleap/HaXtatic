Tags

You include HaXtatic tags in your content sources with a simple notation: often {Prefix|Name|}, sometimes {Prefix|Name: Args|}. (Why curly braces and not <angle brackets>? Easier authoring in all tools and naturally prompter processing.)

Example: {P|date|} includes the current 'page date': 2016-12-18. Supported 'prefixes' or tag types are T, P, B, X:

For a neat overview listing of all your pages/tags.*.html (and/or pages/tags/*.html) articles on this page, check out the |X|hax/repeat: tag type in the HaXtatic docs.
