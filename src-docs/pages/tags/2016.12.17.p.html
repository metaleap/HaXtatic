P-tags: per-page values (date, title, custom..)

{P{..}} tags include (built-in and custom) values specific to the current page.

Built-in

  • {\P{Title}} → current page's first single-line <h1>'s entire inner markup
    — eg. for this page: {P{Title}}
  • {\P{Body}} → current page's entire (mostly already processed) markup source (only usable in {T{ThemeHtmlLink}})
  • {\P{FileName}} → the current page's output file name
    — eg. for this page: {P{FileName}}
  • {\P{OrigPath}} → the current page's (relative local) source path
    — eg. for this page: {P{OrigPath}}
  • {\P{BaseName}} → the current page's name minus file extension and section prefix:
    — eg. for this page named {P{FileName}}: {P{BaseName}}
  • {\P{Date}} and {\P{DateMyFormat}} → current {T{PDate}}
    — eg. for this page: {P{Date}}
  • {\P{PostText}} → current {T{HaxBlokPage}}'s first single-line <p>'s entire inner text
    — eg. for this page: {P{PostText}}

Custom

Define those like {P{%name:value}} in a page; then while that page is being processed, this value will (stripped of leading / trailing white-space) replace all encounters of {P{%name}} anywhere in the output (whether from page, {T{ThemeHtmlLink}} or {T{BlogHtmlLink}} sources), including occurrences produced by {T{HaxTTag}}s or originating from other settings in {T{HaxConfLink}}.

Any characters are acceptable for the name in between {P{% and  :, as well as the value in between  : and  }}.
