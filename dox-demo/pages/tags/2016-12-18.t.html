T-tags: named plain text/markup snippets

{T| .. |} tags directly output named textual contents defined in a *.haxproj file.

{X|h: In a nutshell |}
  • Definitions in {X|lb:haxproj#tdecl *.haxproj|}:
    |T|AuthBlurb: The author is a serial enthusiast
    role-playing <a>on XingedIn</a>.
  • Usage anywhere: {T|AuthBlurb|}
{X|h: See also |}
  • To define page-specific / unique-per-page values (still fully referable from outside the page ie. in {X|lb:tmplsnip templates|}, {X|lb:tmplsnip#snippets snippets|} and {X|lb:haxproj *.haxproj|} files), use {X|lt:p#Custom custom P-tags|}.
  • For more versatile parameterized snippets with named variable value substitutions, {X|lb:haxproj#xdecl declare an X-tag definition|} (or multiple) of type {X|lx:hax.snippet hax.snippet|} instead.
