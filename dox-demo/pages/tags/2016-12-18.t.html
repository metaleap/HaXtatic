T-tags: named plain text/markup snippets

{T| .. |} tags directly output named textual contents defined in a *.haxproj file.

{X|h: In a nutshell |}
  • Definitions in *.haxproj:
    |T|AuthBlurb: The author is a serial enthusiast
    role-playing <a>on XingedIn</a>.
  • Usage anywhere: {T|AuthBlurb|}
{X|h: See also |}
  • To define page-specific / unique-per-page values (still just as referable from outside the page ie. in templates, snippets and *.haxproj files), use custom P-tags.
  • For more versatile parameterized snippets with named variable value substitutions, define X-tags of type hax/snippet instead.
