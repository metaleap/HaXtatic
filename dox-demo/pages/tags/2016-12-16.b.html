B-tags: Blok info (name, title, desc, ..)

A {B| .. |} tag outputs, for the (named or current) {X|lb:bloks Blok|}, the value (as set in a {X|lb:haxproj#bdecl *.haxproj |B| directive|}) of the requested property.

Format: {X|c:{B|propertyname:blokname|}|} or {X|c:{B|propertyname|}|}. In a latter, ie. whenever a blokname is not supplied, the B-tag refers to the "current" (if any) Blok:

  • {B|name|} → name of the current page's associated Blok (if any)
    — eg. for this page: {B|name|}
  • {B|title:name|} → title of the named (or current) Blok
    — eg. for this page, not giving any particular Blok name: {B|title:|}
  • {B|desc:name|} → description of the named (or current) Blok
    — eg. in this site for Blok named basics:
  • {B|atomFile:name|} → relative path to the autogenerated {X|lb:xmlfiles#blokatoms .atom|} file (if any) for the named (or current) Blok
  • {B|blokIndexPageFile:name|} → relative path to the {X|lb:bloks#auto autogenerated|} index page for the named (or current) Blok
    — eg. in this site for Blok named basics: {B|blokIndexPageFile:basics|}
