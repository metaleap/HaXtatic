P-tags: page values (date, title, excerpt, etc..)

{P{...}} tags include (built-in and custom) values specific to the current page.

Predefineds

  • {T{!P}}Title}} → current page's first single-line <h1>'s entire inner markup
    — eg. for this page: {P{Title}}
  • {T{!P}}Body}} → current page's entire (mostly already processed) markup source (only usable in {T{ThemeHtmlLink}})
  • {T{!P}}FileName}} → the current page's output file name
    — eg. for this page: {P{FileName}}
  • {T{!P}}OrigPath}} → the current page's (relative local) source path
    — eg. for this page: {P{OrigPath}}
  • {T{!P}}BaseName}} → the current page's name minus file extension and section prefix:
    — eg. for this page named {P{FileName}}: {P{BaseName}}
  • {T{!P}}Excerpt}} → current page's first single-line <p>'s entire inner text
    — eg. for this page: {P{Excerpt}}
  • {T{!P}}Date}} and {T{!P}}DateMyFormat}} → current {T{PDate}}
    — eg. for this page: {P{Date}}
