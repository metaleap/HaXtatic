Tags

You include HaXtatic tags in your content sources with a simple notation: {Prefix{Name}} or {Prefix{Name: Args}} — for example {P{Date}} includes the current 'page date': 2016/12/18.

(Why curly instead of <angle brackets>? Ease of authoring in all tools and inherently faster processing.) Supported 'prefixes' or tag types are T, P, B, X:

  • T-tags: named plain snippets

    {T{...}} tags directly include named text values that you defined once in your haxtatic.config and then reuse wherever needed.

  • P-tags: page values (date, title, excerpt, etc..)

    {P{...}} tags include (built-in and custom) values specific to the current page.
